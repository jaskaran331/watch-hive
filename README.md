# Watch Hive

Watch Hive is a sleek, premium web application for streaming movies and TV shows, designed with a focus on usability, beautiful UI/UX, and robust performance across devices (Mobile, Desktop, and TV).

## Features
- **Extensive Catalog:** Browse trending, top-rated, and popular movies and TV shows, powered by TMDB.
- **Multiple Video Sources:** Reliable streaming with multiple provider sources.
- **Save & Track Progress:** Save items to your watchlist and track your viewing progress automatically.
- **Responsive Design:** Optimized for all screen sizes, including a dedicated layout for TVs.
- **Golden Theme:** A premium, luxurious dark UI with golden accents.
- **Offline Support:** PWA capabilities to detect network status.

## Tech Stack
- React + Vite
- Context API for state management
- TMDB API for metadata
- CSS variables for theming

## Setup

1. Clone the repository
2. Install dependencies: \`npm install\`
3. Set up your environment variables (using Vercel or locally via \`.env.local\`). You will need a TMDB Read Access Token.
4. Run the development server: \`npm run dev\`

## Disclaimer
Watch Hive does not host any video content. It simply aggregates links from various third-party streaming providers across the internet.
